# 🍂 Choppodoro

A Pomodoro productivity timer built around Tony Tony Chopper from One Piece. Stay focused, take breaks, and let Chopper cheer you on — one session at a time.


---

## ✨ Features

- ⏱️ Customizable focus and break durations
- 🍂 Falling autumn leaves animation
- 🩺 Chopper animations that change based on timer state
- 🎩 Session progress tracker with Chopper hat icons
- 📖 About page explaining the Pomodoro technique
- 🎨 Warm One Piece inspired color palette

---

## 🏴‍☠️ How it works

Choppodoro follows the classic Pomodoro Technique:

1. Set your focus time (default 25 minutes)
2. Set your break time (default 5 minutes)
3. Click **Start** and focus on one task
4. When the session ends, Chopper will let you know it's break time
5. After 4 sessions, take a long break — you earned it!
